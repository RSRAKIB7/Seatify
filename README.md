# 📖 Overview

**Seatify** is a modern Android-based rental management platform designed to simplify the process of finding and managing:

* 🏠 Mess Seats
* 🏢 Hall Rooms
* 🛏️ Hostels
* 🏡 Rental Houses
* 📍 Nearby Accommodation

The project focuses on creating a **fast**, **secure**, and **user-friendly** experience for students, tenants, and property owners.

# 🚀 Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/RSRAKIB7/Seatify.git
```

## 2️⃣ Open in Android Studio

* Open Android Studio
* Click **Open Project**
* Select the Seatify folder

## 3️⃣ Sync Gradle

Wait for Gradle sync to complete.

## 4️⃣ Run Project

Connect an Android device or emulator and click ▶ Run.
